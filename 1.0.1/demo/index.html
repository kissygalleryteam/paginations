Paginations的demo

1.完整版（上下页+中间页码+跳转）

效果
代码 
        var pagination = new Pagination({
            $container: '#page1',
            currentPage: 10,
            totalPage: 15
        }).on('before:skip', function (ev) {
            alert(ev.page);
            //return false; //可以阻止渲染
        }).on('skip', function (ev) {
            alert(ev.page);
        }).on('skip:error', function (ev) {
            alert(ev.page + '不符合');
        });

2.通用版（上下页+中间页码）

效果
代码 
        new Pagination({
            $container: '#page2',
            showSkip: false,
            totalPage: 15
        });

3.简易版（上下页）

效果
代码 
        new Pagination({
            $container: '#page3',
            totalPage: 15,
            showSkip: false,
            showNumbers: false
        });

4.自定义模板版

效果
代码 
        new Pagination({
            $container: '#page4',
            totalPage: 15,
            tpl: customTpl       //XTemplate模块
        });

5.无需组件渲染版（后端渲染+刷新）

效果
< 1 2 3 4 5 6 7 ... > 共 15 页 到第 页
代码 
        new Pagination({
            $container: '#page5',
            currentPage: 1,
            totalPage: 15,
            autoRender: false
        }).on('skip:error', function (ev) {
            alert(ev.page + '不符合');
        });

6.异步加载数据loading版（冻结分页）

效果
代码 
        var pagination = new Pagination({
            $container: '#page6',
            currentPage: 1,
            totalPage: 15
        }).on('skip', function (ev) {
            pagination.set('freeze', true);
            //模拟异步加载
            S.later(function(){
                alert('数据加载完毕');
                pagination.set('freeze', false);
            },2000);
        });
